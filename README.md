# Registration-Application
A simple student or employee registration application developed using java, j2ee, html, css, javascript, jsp and mysql.

## Prerequisties
* Windows OS
* Java JDK and JRE 
* NetBeans IDE
* Wamp Server for MySQL
* Google chrome Browser

## Built using
> Java
>
> HTML
>
> CSS
>
> JavaScript
>
> MySQL
>
> JSP
